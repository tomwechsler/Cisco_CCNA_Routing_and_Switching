# Cisco CCNA Routing and Switching

Prepare for the CCNA exam with comprehensive study materials, including:
- Networking fundamentals
- IP addressing and subnetting
- Routing protocols (OSPF, EIGRP)
- Switching concepts
- VLANs and trunking
- Network security basics
- Troubleshooting techniques
- Hands-on lab exercises (Packet Tracer and GNS3)

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/tomwechsler/Cisco_CCNA_Routing_and_Switching.git
    ```
2. Navigate to the project directory:
   ```bash
   cd Cisco_CCNA_Routing_and_Switching
   ```
3. Explore the study materials and lab exercises.
